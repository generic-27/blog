Closure

Closure in JavaScript refers to when a function is able to rememeber and access its lexical scope even when the function is executing outside is lexical scope. Let's start with an example.
    function foo () {
        var a = 9;

        function bar () {
            console.log (a);
        }

        return bar;
    }

    var baz = foo ();
    baz(); // Closure on bar
In the above example the function bar has closure over the scope of foo. This is because bar is declared inside the scope of foo. The analogy of concentric circles could be used for closures as well because, the lexical scope that forms those concentric circles is closely related to closures. So looking at closures through the lens of the circle analogy, we would look at it inside-out. The inner most scope/circle has closure over all the other scopes/circles. What's fascinating about closure is the persistence of the scope of a function even after execution. Generally when a block of scope finishes execution, it is garbage collected and it's not present in memory, butz closures allow some magic to happen!
Other forms of closure. A function can be passed as a value to another function, and that's code that I have seen several times. All the ways in which a function can be passed around as values and invoked in other locations, are all examples of closures. Example below!
    function foo () {
        var a = 2;

        function bar () {
            console.log(a);
        }

        baz(bar);// passed as value
    }

    function baz (fn) {
        fn(); // Closure here!
    }
Closures are everywhere and one of the best examples is the setTimeOut function.
    var a = 10;
    setTimeout(function time() {
        console.log(a);
    }, 1000);
In the above setTimeout example, the functiontimer has a scope closure over the global scope, it rememebers the value a. We pass the function timer as a parameter to the setTimeout, which is a functionReference and that function has the closure over the global scope.
Modules
Modules are the of the prominent and powerful examples of closures.
    function moduleExample() {
        var a = 10;
        var b = 5;

        function printA() {
            console.log(a);
        }

        function printB() {
            console.log(b);
        }

        return {
            printA: printA,
            printB: printB
        };
    }

    var m = moduleExample();

    // closure over the variables
    // a and b
    m.printA();
    m.printB();
Modules are functions itself and this example is in a way similar to the fist closure example. The only difference being that in the case of a module it returns an object with multiple functions that have closure over the scope of the module function.
